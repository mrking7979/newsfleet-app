# 📰 NewsFleet App

Welcome to **NewsFleet** 🚀  
A simple and beginner-friendly React app.

---

# 🎯 What is this app?

This app is like a **mini news dashboard** 📰

You can:
- ✍️ View articles  
- ✅ Approve articles  
- 💬 Add comments  

---

# 🧠 Technologies Used

- ⚛️ React  
- 🟦 TypeScript  
- 🧪 Jest (Testing)  
- 🧹 ESLint (Code Quality)

---

# 🚀 How to Run This Project (Step-by-Step)

## 🧾 Step 1: Download the project

Click **Code → Download ZIP**  
OR use git:

git clone https://github.com/mrking7979/newsfleet-app.git

---

## 📂 Step 2: Open folder

cd newsfleet

---

## 📦 Step 3: Install packages

npm install

---

## ▶️ Step 4: Start the app

npm start

👉 Now open browser:  
http://localhost:3000

---

# 🧪 Run Tests

npm test

👉 Press **a** to run all tests

---

# 🧹 Check Code (Lint)

npm run lint

---

# 🎨 Format Code

npm run format

---

# 🧑‍💻 How to Use the App

1. Open the app  
2. Click **Approve button**  
3. See → "Approved!" ✅  
4. Write a comment  
5. Click **Post**
